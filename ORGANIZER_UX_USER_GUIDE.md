## Creating Quizzes

### Starting a New Quiz

1. Click **[+ New]** on your dashboard
2. Choose creation method:
   - **Start from Blank**: Create from scratch
   - **Use Template**: Select from your templates

3. Enter quiz details:
   - **Name**: Give your quiz a descriptive name
   - **Topic** (optional): Categorize your quiz

   - **Privacy**: Choose Private or Public

### Quiz Creation Options

**From Blank**:
- Empty quiz ready for questions
- Full creative control
- Best for unique content

**From Template**:
- Pre-populated with questions
- Edit before starting
- Saves time for recurring quizzes

### Adding Questions

1. Click **[+ Add Question]**
2. Enter your question text
3. Add answer options (minimum 2)
4. Mark the correct answer(s)
5. Click **[Save]**

**Tips**:
- Use clear, concise question text
- Provide 2-4 answer options
- Mark at least one correct answer
- Preview how it looks to participants

### Reordering Questions

- **Drag and Drop**: Click and drag questions to reorder
- Questions are numbered automatically
- Changes save immediately

### Editing Questions

1. Click on a question in the list
2. Modify text or answers
3. Click **[Save]** to update

### Deleting Questions

1. Click on a question
2. Click **[Delete]** button
3. Confirm deletion

### Preview Mode

- Click **[Preview]** to see participant view
- Test how questions appear
- Check answer button layout
- Exit preview to continue editing

### Saving as Draft

- Click **[Save Draft]** anytime
- Your progress is saved
- Return later to continue editing
- Auto-save happens every 30 seconds

### Starting Your Quiz

1. Add at least one question
2. Click **[Ready to Start Quiz]**
3. Review the confirmation dialog:
   - Game PIN displayed
   - Join instructions shown
   - Participant link provided
4. Click **[Start Quiz]** to go live

**Requirements**:
- âœ“ At least one question
- âœ“ Each question has 2+ answers
- âœ“ Each question has a correct answer

## Templates

Templates let you save quizzes for reuse, saving time when creating similar content.

### Creating a Template

**From Existing Quiz**:
1. Open a quiz in Setup Mode
2. Click **[Save as Template]**
3. Enter template details:
   - **Name**: Descriptive template name
   - **Description**: What the template contains
   - **Public**: Share with other organizers (optional)
4. Click **[Save Template]**

**Template Information**:
- Saves all questions and answers
- Preserves question order
- Does not save participant data
- Can be edited after creation

### Using Templates

**Creating from Template**:
1. Click **[+ New]** on dashboard
2. Select **Use Template**
3. Browse your templates
4. Click on a template to select
5. Enter new quiz name
6. Click **[Create]**

**Template Preview**:
- Hover over template cards to preview
- Shows question count
- Displays topic and description
- View creation date and usage count

### Managing Templates

**Your Templates**:
- Access via Template Selector
- Edit template details
- Delete unused templates
- View usage statistics

**Public Templates**:
- Browse templates shared by others
- Filter by topic
- Search by name or description
- Use any public template

### Template Best Practices

âœ“ **Descriptive Names**: Use clear, searchable names
âœ“ **Add Descriptions**: Explain what the template covers
âœ“ **Organize by Topic**: Use consistent topic categories
âœ“ **Update Regularly**: Keep templates current
âœ“ **Share Quality Content**: Only make high-quality templates public

## Privacy Controls

Control who can discover and join your quizzes with privacy settings.

### Privacy Options

**Private Quiz** ğŸ”’:
- Default setting
- Requires Game PIN to join
- Not visible in public browser
- Best for: Classes, private events, controlled groups

**Public Quiz** ğŸŒ:
- Discoverable by anyone
- Appears in public quiz browser
- Still requires Game PIN (displayed publicly)
- Best for: Open events, community quizzes, public competitions

### Setting Privacy

**During Creation**:
1. Select privacy when creating quiz
2. Choose Private or Public
3. Can be changed before starting

**Before Starting**:
1. Open quiz in Setup Mode
2. Use Privacy Selector dropdown
3. Select desired privacy level
4. Save changes

**Restrictions**:
- Cannot change privacy once quiz is live
- Privacy setting persists after completion

### Privacy Indicators

- ğŸ”’ Lock icon: Private quiz

## Public Quiz Discovery

Participants can now discover and join public quizzes without needing a direct invitation.

### Public Quiz Browser

**For Participants**:
1. Navigate to **Browse Public Quizzes**
2. View available public quizzes
3. Filter by status (Live, Upcoming)
4. Search by name or topic
5. Click **[Join Quiz]** to participate

**Quiz Information Displayed**:
- Quiz name and topic
- Current participant count
- Status (Live or Upcoming)
- Description
- Game PIN

### Making Your Quiz Discoverable

1. Set quiz to **Public** privacy
2. Add a descriptive **Topic**
3. Write a clear **Description**
4. Start the quiz to make it live
5. It appears in the public browser automatically
