## How to Get Started

### 1. Explore Your Dashboard

1. Click **"My Quizzes"** in the navigation
2. See all your quizzes organized by status
3. Try the search and filter features
4. Click on a quiz to manage it

### 2. Create Your First Template

1. Open an existing quiz (or create a new one)
2. Click **"Save as Template"**
3. Give it a descriptive name
4. Choose whether to make it public
5. Use it to create new quizzes instantly!

### 3. Try Setup Mode

1. Create a new quiz
2. Add questions in the clean Setup Mode interface
3. Drag to reorder questions
4. Preview how they look
5. Click "Ready to Start" when done

### 4. Run a Quiz in Live Mode

1. Start your quiz from Setup Mode
2. Share the Game PIN with participants
3. Watch the focused Live Mode interface
4. See real-time updates as participants join and answer
5. Use the highlighted controls to advance

### 5. Make a Quiz Public

1. Create or edit a quiz
2. Set Privacy to "Public"
3. Add a topic and description
4. Start the quiz
5. It automatically appears in the public browser!

## Frequently Asked Questions

### General

**Q: Do I need to do anything to access these features?**  
A: No! All features are automatically available. Just log in and explore.

**Q: What happened to my existing quizzes?**  
A: They're all in your new dashboard, categorized as "Past" quizzes. All data is preserved.

**Q: Can I still use the old workflow?**  
A: The new workflow is designed to be intuitive and familiar. Give it a try - we think you'll love it!

### Dashboard

**Q: How do I find a specific quiz?**  
A: Use the search bar at the top of the dashboard. Type the quiz name or topic.

**Q: Can I delete old quizzes?**  
A: Yes! Click on a quiz card and select "Delete" from the options.

**Q: How do I see only live quizzes?**  
A: Click the "Live" tab at the top of the dashboard.

### Templates

**Q: Can I edit a template after creating it?**  
A: Templates are snapshots. Create a quiz from the template, edit it, and save as a new template.

**Q: Are my templates private?**  
A: Yes, unless you choose to make them public when creating.

**Q: Can I use someone else's public template?**  
A: Absolutely! Browse public templates and use any you like.

### Privacy

**Q: Can participants see my private quizzes?**  
A: No. Private quizzes require the Game PIN and aren't listed publicly.

**Q: Can I change privacy after starting a quiz?**  
A: No. Privacy is locked once a quiz goes live for security reasons.

**Q: What's the difference between private and public?**  
A: Private quizzes require the Game PIN. Public quizzes appear in the public browser but still use Game PINs.

### Setup & Live Modes

**Q: Can I edit questions during a live quiz?**  
A: No. Questions must be finalized in Setup Mode before starting.

**Q: Can I switch back to Setup Mode?**  
A: No. Once live, the quiz stays in Live Mode until completed.

**Q: What if I need to change a question mid-quiz?**  
A: End the quiz, create a new one with corrections, and restart.
